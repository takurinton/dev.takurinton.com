# {{ $frontmatter.title }}

完全にポエム。  
大した内容じゃないけど別に組織や人間を否定するわけではなくて、シンプルに自己嫌悪になった結果の殴り書きスペース。  

俺は人生ここまで後悔だらけだった。  
あの時ああしてればよかったなという選択の後悔や、もっと勉強しておけばよかったみたいな後悔が多い。   
  
それが特にこの半年は特に大きかった。  
一番大きいのは、React が書けてればなということ。  
これは、仕事で React を書くことができていたら、俺のフロントエンドの技術力は全然違っていたんだろうなみたいなことを想像してしまった。  
正直、PHP は楽しくない。めちゃくちゃ正直に言うと、今の部署のサービスにみじんも興味がない。プロダクト的にも技術的にも興味がない場所で仕事をしたらどうなるか？答えは簡単、虚無になる。  
OJT を担当してくれる方はいい人、部署の人もとても良い人、なのに事業や技術に興味を持てない、なかなか楽しんでコードを書けない自分が嫌になる。  
コードを読む力は身についた。こういうコードがあるんだ、こういう書き方があるんだという知識も身についた。ただ、興味のない知識が身についても楽しくない。  
なんか、完全に技術は手段の状態になっていた。俺は技術は目的でもある、そう思いたい人間だ。だからこそ辛かった。  
  
俺はフロントエンドがやりたい。フロントエンドがやりたいけど、できてない。  
フロントエンドだろうがサーバサイドだろうがコードを書くという点で見れば変わらないだろう、それはそう。ただ、モチベーションや使用ツール、フレームワークやライブラリ、CDN やキャッシュについてなど、全然違うと思う。コーディングとかいうクソでかなくくりではなくて、シンプルにフロントエンドが書きたい、React で UI を作りたい、パフォーマンスにこだわりたい。そう思ってる気持ちに対して仕事の内容がまったくの別のものなので辛くなってしまった。  
  
そして、身近な人の存在も大きく影響している。別に敵に回すわけではないけど、俺のパートナーはフロントエンドを書いてる。喉から手が出るほど羨ましい。俺の友人はフロントエンドを書いている。プロトコルの知識や周辺ツールの知識も豊富。死ぬほど羨ましい。  
それに対して自分はどうだろうか？半年間、PHP だけを書いてきた。たまに jQuery も書いたけど、基本的には PHP、俺は何をしてるんだ？若い時のリソースをこんなところに回していいのか？そうなる。  
そして個人開発でむりに React を書いてみる。当然書けない、事前知識がないんだから。一般的な書き方も知らない、設計についても知らない、パフォーマンスについても、バンドルについても知らない。  
みんなが当たり前に知ってるもの、使ってるものに対しての知識がなさすぎる。仕事をしてれば身につくというものではないことはわかってる。ただ、個人開発だと限界があるんだよ。  
自作 VDOM ライブラリを作ってみたことも、バンドルツールを作ってみたこともあった。けどそんなのどうでもいい、俺は仕事で React を書いて、ユーザーに UI を届けたい、みんなに追いつきたい、それだけなんだよ。  
仕事以外の時間でフロントエンドを書くようになってから生活習慣が崩れた。睡眠サイクルが乱れた。メンタルも壊れた。キャパオーバーになるのは自明だった。  
  
フロントエンドの実務経験がない、それがこの半年めちゃくちゃコンプレックスだった。  
別に会社が嫌いなわけでもなければ、サーバーサイドが嫌いとかいうわけでもない。ただ、フロントエンドが書きたいだけ。  
ここ最近の鬱の理由は半分くらいここら辺が原因だと思う。仕事をする時間が増えて、仕事中心の生活になってきたからこそ思う。

こういう思考になってしまう自分が嫌だ。  
フロントエンドが書きたいが故にサーバサイドに対して楽しさを見出せなくなる自分も、全て悲観的に捉えてしまう自分も、全部嫌だ。  
いつからか知らないけど、俺はプログラミングが楽しいと感じなくなってしまった。  
だが、楽しくなくてもエンジニアとして就職をする以上、コードは書かなければいけない。  
俺はこうして惰性でコードを書き続け、適当に人生を終えるんだろうな。  
  
こんな人生つまらないので早く終わらせたいけど、俺に死ぬ勇気はない。消えてしまいたいと思うけど、手取り早く消えられる方法などない。  
ボタンを押したら消えられる、そんな世の中になるといいね。
